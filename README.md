# 😈 DemoniDraw (v1.0 - Edición Demoníaca)

DemoniDraw es un potente motor de automatización por gestos táctiles para Android con acceso Root. A través de un servicio flotante invisible de baja latencia, la aplicación permite al usuario dibujar trazos libres en cualquier parte de la pantalla ("Triggers visuales") para ejecutar de forma instantánea macros complejas, scripts de consola, control de procesos e inyección de eventos táctiles a bajo nivel.

---

## ✨ Características Principales

* **🎨 Reconocimiento de Gestos sobre la UI:** Lienzo flotante interactivo (Overlay) que captura trazos en tiempo real sobre cualquier aplicación o juego sin interrumpir el flujo del sistema.
* **⚡ Automatización por Root Integrada (Macros):**
    * Mapeo de dibujos sencillos (letras, formas, curvas) para desencadenar secuencias multitarea complejas.
    * Inyección precisa de coordenadas (`input tap`, `input swipe`) y comandos de sistema (`am start`, `am force-stop`).
    * Capacidad para automatizar flujos completos en aplicaciones de terceros (ej. abrir chats específicos, enviar contenidos y emular interacciones táctiles en bucle).
* **📈 Rendimiento Optimizado:** Algoritmo ligero de reconocimiento de patrones diseñado para minimizar el uso de CPU y el impacto en la batería durante la escucha activa.
* **🛡️ Integración Segura con Superusuario:** Totalmente compatible con binarios Root modernos a través de Magisk o KernelSU para la ejecución fluida de comandos `su`.

---

## 🛠️ Stack Técnico

* **Ecosistema:** Android SDK, ventana flotante con permisos de accesibilidad/superposición (`SYSTEM_ALERT_WINDOW`).
* **Seguridad:** Protección y optimización de código en producción mediante técnicas avanzadas de ofuscación contra ingeniería inversa.
